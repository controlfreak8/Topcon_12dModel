Create linework

With the desired data in a view i.e. :

  1. Survey => Topcon => Write LN3 file

  1. If the strings are occluded under a tin, send 'Tin/Raster to back'

  1. Select the desired data by view as appropriate
  2. Choose an appropriate 'Layer name', 'Layer colour' and 'Internal name'
  3. Populate a name for the file and select an output directory
  4. Write

NB: The selected data from the model will be exported with the layer name specified in the 'Layer name' text. This is the field that is used to inform the model name if the process were performed in the opposite direction i.e. from LN3 back to 12d Model.

A 'LN3' file is linework, while a TN3 file is a tin.
